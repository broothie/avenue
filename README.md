# Avenue

**Avenue** is a nesting-based HTTP router, with support for [OpenAPI](https://swagger.io/docs/specification/about/) 
spec generation. 

Avenue uses [gorilla/mux](http://www.gorillatoolkit.org/pkg/mux) under the hood.
